# Slavia Furtado - Portfolio

Personal portfolio website built with React and Tailwind CSS.

## Stack

- React
- Tailwind CSS
- React Scroll (smooth navigation)
- React Icons
- Getform.io (contact form)

## Sections

- **Home** - intro and CTA
- **About** - background and tech stack
- **Skills** - technologies and tools
- **Projects** - selected work
- **Contact** - contact form

## Getting Started

```bash
npm install
npm start
```

## Deployment

Built for static hosting - works with GitHub Pages.
